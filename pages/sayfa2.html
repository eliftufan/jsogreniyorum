JavaScript'e Giriş

JavaScript, yaygın olarak web tarayıcılarında kullanılan dinamik bir script dilidir. Etkileşimli ve dinamik web sayfaları oluşturmaya imkan sağlar.

HTML'de, JavaScript kodu "script" ve "/script" tagleri arasına eklenmelidir. JavaScript dosyalarının uzantısı .js şeklindedir.

Javascript hem çift tırnağa hem tek tırnağa duyarlıdır.

Javascript özelliklerini sayacak olursak; HTML içeriğini değiştirebilir, özellik değerlerini değiştirebilir, HTML stillerini değiştirebilir, HTML öğelerini gizleyebilir ve gösterebilir.
