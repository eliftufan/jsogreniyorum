Javascript DOM (Document Object Model)

DOM Document Object Model kelimesinin kısaltılmasından gelir ve Türkçe karşılığı Belge Nesne Modelidir.

DOM, HTML, XML gibi belgelerin diğer programlama dilleri veya script dilleriyle iletişim kurabilmesini sağlamak için geliştirilmiş bir arabirimdir.

Yukarıdaki görselde olduğu gibi HTML sayfasında bulunan "head", "body", "h1", "img" gibi etiketler(tagler) birer DOM nesneleridir.

DOM sayesinde HTML veya XML elemanları ile programlama veya script dilleri arasında haberleşme sağlayarak daha etkileşimli web siteler yapılabilmesini sağlanır.

DOM Metodları:

DOM metodları, gerçekleştirebileceğimiz eylemlerdir.

getElementedById Metodu:

Bir HTML etiketine erişmek için kullanılan en sık metoddur.Etiketin id özelliğini kullanır.

HTML Öğeleri Bulma

  • document.getElementById(id):id niteliği ile eşleşen ilk kaydı getirir.

  • document.getElementsByName(name):neme niteliği ile eşleşen tüm kayıtları dizi olarak getirir.

  • document.getElementsByTagName(etiketdı):etiket adı ile eşleşen tüm kayıtları dizi olarak getirir.

  • document.getElementsByClassName(classAdı):class niteliği ile eşleşen tüm kayıtları dizi olarak getirir.

  • document.querySelector(seçiçi):seçiçi ile eşleşen ilk kayıtları dizi olarak getirir.

  • document.querySelectorAll(seçiçi):seçiçi ile eşleşen tüm kayıtları dizi olarak getirir.

HTML Etiketleri Düzenleme:

  • etiket.innerHTML:Etiketin içini okuma ve değiştirme için kullanılır.

  • etiket.attribute:Etiketin niteliğin değiştirmek için kullanılır.

  • etiket.setAttribute(nitelik,değeri):Etiketin niteliğini değiştirmek için kullanılır.

  • etiket.style.property:Etiketin stilini değiştirmek için kullanılır.

  • etiket.classList:Etiketlerin class niteliği üzerinde silme,değiştirme,ekleme,class listesini öğrenme gibi işlemleri yapmak için kullanılır.

HTML Etiketi Ekleme ve Silme:

  • document.createElement(element):HTML etiketi oluşturma.

  • document.removeChild (element):HTML etiketi silme.

  • document.appendChild(element):HTML etiketi ekleme.

  • document.replaceChild(element):HTML etiketini değiştirme.
