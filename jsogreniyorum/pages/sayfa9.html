JavaScript

DİZİLER

Dizileri, birden çok değeri tek bir değişkende depolamak için kullanılır.

var cars = ["Saab", "Volvo", "BMW"];

Diziyi oluşturma:

var array_name = [item1, item2, ...];

Dizinin elemanlarına ulaşmak:

Bir dizi öğesine dizin numarasına bakarak erişebilirsiniz.

var name = cars[0];

var cars = ["Saab", "Volvo", "BMW"];
document.getElementById("demo").innerHTML = cars[0];

Dizi Öğesini Değiştirme

cars[0] = "Opel";

Dizi Özellikleri ve Yöntemleri

length Özelliği:

Bir dizinin length özelliği, bir dizinin uzunluğunu (dizi öğesi sayısı) döndürür.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.length; // dizinin boyutunu 4 olarak döndürür

İlk Dizi Öğesine Erişim:

fruits = ["Banana", "Orange", "Apple", "Mango"];
var birinciEleman = fruits[0];

Son Dizi Öğesine Erişme:

fruits = ["Banana", "Orange", "Apple", "Mango"];
var last = fruits[fruits.length - 1];

Dizi Metodları:

Dizileri Dizelere Dönüştürme:

ToString () JavaScript yöntemi, bir diziyi (virgülle ayrılmış) dizi değerlerinin dizesine dönüştürür.

var fruits = ["Banana", "Orange", "Apple", "Mango"];
document.getElementById("demo").innerHTML = fruits.toString(); // Banana,Orange,Apple,Mango haline dönüştürür.

Pop() ve push() metodları:

Pop () yöntemi, bir dizideki son öğeyi kaldırır:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.pop(); // En son elemanı olan "Mango" yu kaldırır.

Push () yöntemi, bir diziye (sonunda) yeni bir öğe ekler:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.push("Kiwi"); // "Kiwi" elemanını diziye ekler.

Shifting Özelliği

Shift () yöntemi, ilk dizi öğesini kaldırır ve diğer tüm öğeleri daha düşük bir dizine "kaydırır".

Splice() Metodu:

Bir diziye yeni öğeler eklemek için splice () yöntemi kullanılabilir:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.splice(2, 0, "Lemon", "Kiwi");

Dizilerde Sıralama:

Sort () yöntemi, bir diziyi alfabetik olarak sıralar:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.sort();

Diziyi Ters Çevirme:

Reverse () yöntemi, bir dizideki öğeleri tersine çevirir.Bir diziyi azalan sırada sıralamak için kullanabilirsiniz:

var fruits = ["Banana", "Orange", "Apple", "Mango"];
fruits.sort(); // Diziyi sıralarız.
fruits.reverse(); // Diziyi tersten sıralar.
