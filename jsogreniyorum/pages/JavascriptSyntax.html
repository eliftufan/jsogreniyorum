JavaScript Syntax(Sözdizimi) Yapısı

JavaScript sözdizimi, JavaScript programlarının nasıl oluşturulduğu ile ilgili kurallar dizisidir.

Javascript Değerleri

Syntax iki tür değer tanımlar. Birincisi sabit (değişmez) değerler, ikincisi ise değişken değerlerdir.

Sabit(değişmez) Değerler

Sabit değişkenler sayılardır. Ondalıklı ve tam sayı şeklinde yazılırlar.Örneğin;
10.5
10
şeklindedir.

Değişken Değerler

Veri değerlerini saklamak için değişken değerleri kullanılır."var" ibaresiyle tanımlanırlar."=" ile değerleri atanır.Örneğin;
var x;
x = 0
şeklindedir.

Dizeler çift tırnak("") veya tek tırnak('') içerisinde yazılırlar.

JavaScript Operatörleri

Hesaplamak için aritmetik işleçleri ( +, -, *, /, %) kullanılır.
Değişkenlere değer atamak için bir atama operatörü ( =) kullanılır.

Daha detaylı olarak değişkenler ve operatörler bölümünde göreceğiz.

Yorum İfadeleri

Javascript'te yorum ifadeleri " // " veya " /*....*/ " şeklindedir.

JavaScript Büyük/Küçük Harfe Duyarlıdır!

Değişkenleri tanımlarken çok dikkatli olmalıyız. lastName ile LastName farklı değişkenlerdir.

DİKKAT!

Bir değişken birden fazla kelime içeriyorsa şöyle yapmalıyız:
-Birinci olarak altçizgi kullanabiliriz. Örneğin; name_surname şeklinde.
-İkinci olarak ilk harflerini büyük yazarız diğer harflerini küçük yazarız. Örneğin; NameSurname şeklinde.
-Üçüncü olarak ise sadece ikinci kelimenin baş harfini büyük yazarız. Örneğin; nameSurname şeklinde.
